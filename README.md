# Credit AI Engine

AI-powered credit dispute and optimization engine with:

- Automated strategy selection
- Evidence analysis
- Adaptive learning and outcome feedback loops

## Setup

1. Clone repo
2. Install dependencies: `npm install`
3. Add `.env` file with Supabase keys
4. Run backend: `node server.js`

## Features

- Supabase database integration
- AI-driven dispute strategies
- API endpoint `/analyze` for predictions
